# Map Principles

## Selection clarity

- Selected parcels should stand out immediately.
- Hover and click feedback should be obvious on the first interaction.
- A user should always understand which project the map is currently telling the story about.

## Camera behavior

- Start broad enough to communicate market context.
- Tighten framing after selection to make the project feel important.
- Avoid disorienting spins or novelty motion.

## Layer discipline

- Keep the base dark and unobtrusive.
- Use a small number of accent colors.
- Resist the urge to add more data just because the map can hold it.

## Story sequence

The ideal conversation is:

1. macro market context
2. selected opportunity
3. why this parcel or district matters
4. what the investor gets access to next
